## 🎮 What You'll See

### Game 1: DOMES
Build a dome around one person using the entire US government.  
**Score:** Cosm (weakest dimension = total)

### Game 2: SPHERES  
Activate public spaces in cities.  
**Score:** Chron (m² × time)

### Production Pipeline
1. Development
2. Pre-Production
3. Production
4. Post-Production
5. Distribution

## 🏗️ Architecture

- **Frontend:** React + Vite (embedded)
- **Backend:** FastAPI (Python)
- **Port:** 9000 (serves both API and UI)
- **Storage:** In-memory (SQLite ready)
